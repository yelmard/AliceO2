Detailed Description

template<class RootContainer>
class o2::experimental::histhelpers::HistContainer< RootContainer >

Container class for storing and saving root histograms of any type. RootContainer (TObjArray, TList or TFolder) inheritance is used to interface with O2 file writing functionality.
