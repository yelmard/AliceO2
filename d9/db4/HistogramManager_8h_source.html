54  // Create a new histogram in the class <histClass> with name <name> and title <title>
55  // The type of histogram is deduced from the parameters specified by the user
56  // The binning for at least one dimension needs to be specified, namely: nXbins, xmin, xmax, varX which will result in a TH1F histogram
57  // If the value for a variable is left as -1 then that is considered to not be used
58  // Up to 3 dimensional histograms can be defined with this function
59  // If isProfile = true, the last specified variable is the one being averaged
60  // For the case of TProfile3D, the user must use the varT to specify the averaged variable
61  // If specified, varW will be used as weight in the TH1::Fill() functions.
62  // If specified, the xLabels, yLabels, zLabels will be used to set the labels of the x,y,z axes, respectively.
63  // The axis titles will be set by default, if those were specified (e.g. taken from a Variable Manager)
64  // Otherwise these can be specified in the title string by separating them with semi-colons ";"
